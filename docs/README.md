## 🚀 Inicio Rápido

### Instalación Básica
```bash
git clone <repo>
cd flare-df
cp .env.example .env
chmod +x start.sh
./start.sh
```

### Primeros Pasos
1. Lee [QUICKSTART.md](./QUICKSTART.md) para usar la app en 5 minutos
2. Consulta [INSTALLATION.md](./INSTALLATION.md) si necesitas ayuda con requisitos
3. Revisa [EXAMPLES.md](./EXAMPLES.md) para casos de uso reales
